# 一、Vue的生命周期

```markdown
1. Vue的生命周期是一个简称，又名生命周期回调函数，生命周期函数，生命周期钩子

2. Vue的生命周期就是'Vue在特殊时刻调用的一些特殊名称的函数'，这些函数的名称是Vue规定好的，并且都是Vue实例的一个配置项。

3. 生命周期函数中的this是vm或者是组件对象
```

![](img/生命周期.png)

```markdown
1. mounted:发ajax请求,启动定时器,绑定自定义事件,订阅消息等初始化操作


2. beforeDestroy:清除定时器,解绑自定义事件,取消订阅消息等收尾工作.


3. 关于销毁Vue实例
	*销毁后借助Vue开发者工具看不到任何信息
	
	*销毁Vue实例后自定义事件会失效,但是原生的DOM事件依然有效
	
	*一般不会在beforeDestroy操作数据,因为即使数据改变了,也不会触发Vue的更新流程,更别说重新渲染页面了,所以没有意义.



**********遇到一个坑：编辑一段文字，点击文字切换为input，并默认获取焦点。但是获取焦点的代码不生效**********

** 是因为写的逻辑与获取焦点的代码执行完成之后才重新解析模板，聚焦当然不生效

** updated钩子函数：可以解决这个问题，把聚焦的代码放在钩子里，解析完成执行updated

** $nextTick(回调)：这个函数的意思是告诉Vue，在代码执行完成模板解析之后执行$nextTick的回调，所以想要它执行就需要修改数据来重新解析模板

** $nextTick使用场景：改变数据后，基于已经更新的DOM进行操作时使用




```
