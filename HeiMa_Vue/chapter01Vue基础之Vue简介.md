# 一、Vue简介

## ①Vue是什么

```markdown
1. Vue是一个渐进式的JS框架

2. 渐进式怎么理解：
    *简单应用：只需要一个简单小巧的核心库
    *复杂应用：可以引入Vue的各种组件库
    *从小巧的核心库逐渐递进到使用各式各样的Vue组件库
```

## ②Vue的特点

```markdown
1. 采用组件化开发的思想
	*把一个页面划分为各个模块，每个模块为一个组件，也就是一个.vue文件，这个文件中会写html，css，js。封装了代码，提高了代码的复用率
	
2. 使用声明式编码
	*命令式编码：类似于面向过程的编码方式，代码一行一行执行，有一行代码就执行一行，关注点是流程，但凡缺了一个流程，结果就不正确
	
	
	*声明式编码:关注的是目标，而不是流程。它告诉计算机的是去做什么，而不是如何做
	
3. 使用虚拟DOM + Diff算法，尽量复用DOM节点
	*原生的js程序，得到数据之后会直接加入到页面dom中，当下一次获取的数据在原来数据的基础上添加了几条的时候，还是会把数据全部放到页面dom中，这样就造成了一个问题，原数据和新数据相比有些数据是一样的，但是还是把旧dom移除了，换上了新的dom
	
	*使用虚拟dom：得到数据之后，会把数据放到虚拟dom中，然后虚拟dom把数据放到真实的页面dom，新数据来了之后，会产生一个新的虚拟dom，新的虚拟dom和旧的虚拟dom做比较（diff算法），看哪些数据是相同的，相同的数据，在真实dom上则复用，不同的数据，则由虚拟dom插入到真实dom中，这样就复用dom了
	
```
